# Inventro - Sistema de Gestão de Estoque

Sistema moderno e responsivo de gestão de estoque construído com PHP e CodeIgniter.

## 🌟 Recursos Principais

- **Gestão de Itens** - Adicione, edite e gerencie produtos de forma eficiente
- **Gestão de Fornecedores** - Controle completo de fornecedores e transações
- **Gestão de Clientes** - Cadastro e acompanhamento de clientes
- **Compras** - Rastreamento de compras e análise de lucros/perdas
- **Faturas** - Geração e gestão de faturas
- **Controle de Estoque** - Monitoramento robusto de inventário
- **Contabilidade** - Sistema integrado de gestão financeira
- **RH** - Gestão de funcionários, presença e folha de pagamento
- **Relatórios** - Relatórios detalhados de vendas, compras e estoque
- **Permissões** - Controle de acesso baseado em funções

## 📋 Requisitos do Sistema

- PHP 7.4 ou superior
- MySQL 5.0.12 ou superior
- Apache 2.4.18 ou superior (ou Nginx/IIS)

## 🚀 Instalação

1. Extraia o arquivo `inventro_main.zip`
2. Faça upload para o servidor via cPanel ou FTP
3. Configure o banco de dados
4. Acesse `http://seudominio.com/install`
5. Siga as instruções do instalador
